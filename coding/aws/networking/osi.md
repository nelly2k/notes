# Layers of OSI model
Open Systems Interconnection
1. Physical - CAT5, fiber optic cable
2. Data Link - MAC
3. Network - IP, ARP
4. Transport - TCP
5. Session - Setup, Negotiation, Teardown
6. Presentation - TLS/SSL, Compression
7. Application - Web Browser
